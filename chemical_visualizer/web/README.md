# Chemical Equipment Parameter Visualizer - Web Frontend

React.js web application for visualizing chemical equipment data.

## Features
- CSV file upload
- Interactive charts (Chart.js)
- Data table view
- Summary statistics
- PDF report download
- Authentication (Basic Auth)
- History of last 5 datasets

## Setup

1. Install dependencies:
```bash
npm install
```

2. (Optional) Configure environment:
```bash
copy .env.example .env
```

3. Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Configuration

Set the backend API URL in `.env`:
```
REACT_APP_API_URL=http://127.0.0.1:8000/api
```

## Available Scripts

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm test`

Launches the test runner in the interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The optimized build will be in the `build/` directory, ready for deployment.

## Technologies
- React 19
- Chart.js
- Axios
- Modern CSS Grid Layout
