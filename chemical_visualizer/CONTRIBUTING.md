# Contributing to Chemical Equipment Parameter Visualizer

Thank you for considering contributing to this project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/chemical_visualizer.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes
6. Commit: `git commit -m "Add: your feature description"`
7. Push: `git push origin feature/your-feature-name`
8. Create a Pull Request

## Development Guidelines

### Backend (Django)
- Follow PEP 8 style guide
- Add tests for new features
- Update API documentation
- Use type hints where appropriate

### Frontend (React)
- Use functional components with hooks
- Follow ESLint rules
- Maintain responsive design
- Add comments for complex logic

### Desktop (PyQt5)
- Maintain consistency with web UI
- Handle errors gracefully
- Test on multiple platforms

## Testing

### Backend Tests
```bash
cd backend
python manage.py test
```

### Frontend Tests
```bash
cd web
npm test
```

## Code Review Process

1. All submissions require review
2. Code must pass existing tests
3. New features should include tests
4. Documentation must be updated

## Reporting Bugs

Use GitHub Issues with:
- Clear title
- Steps to reproduce
- Expected vs actual behavior
- Screenshots (if applicable)
- Environment details

## Feature Requests

Submit via GitHub Issues with:
- Use case description
- Proposed solution
- Alternative solutions considered

## Questions?

Open a GitHub Discussion or Issue for any questions.
